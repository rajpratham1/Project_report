# Project Website: Breast Cancer Detection using Neural Networks

This repository contains the source code for a static website presenting a project report on "Breast Cancer Detection using Neural Networks in PyTorch".

## Description

The website provides a detailed, professional project report that includes:
- Abstract
- Introduction
- Objectives
- Dataset Description
- Methodology and Implementation Steps
- Model Architecture
- Results and Evaluation Metrics
- Visualizations
- Conclusion
- Future Scope
- References

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to View

Simply open the `index.html` file in a web browser to view the website.
